What is this? 
The included XML files contain every page in Smutty Sex Romp (as of 28 May 2020), and each page's revision history. They were created using the Special:Export function in MediaWiki 1.15.1

What can I do with it? 
XML dumps can be imported into a suitable MediaWiki. See https://www.mediawiki.org/wiki/Manual:Importing_XML_dumps. To create a complete copy, every XML file from both folders ('Locations' and 'Pages, Category, Templates') must be imported. 

Disclaimer: 
Because the integrity of the Special:Export function cannot be guarunteed, it's possible that pages will not import properly, or will not import at all. 

What are the word documents for? 
I've included a list of all article titles for the XML files in their respective folders. Each list is formatted for the Special:Export function (http://editthis.info/create_your_own_story/?title=Special:Export), and can be directly copy-pasted there to create XML files corresponding to those page titles. Editthis.info, as of 05/28/2020, will not create very large XML files-- the list contained in 'Pages, Category, Templates' must be broken into smaller pieces. For more information on creating your own backups this way, see http://editthis.info/wiki/Help:Backing_up_your_wiki
